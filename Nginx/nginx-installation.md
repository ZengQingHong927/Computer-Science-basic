更新 apt-get ppa nginx 版本

* sudo add-apt-repository ppa:nginx/stable 
* sudo apt-get update
* sudo apt-get install nginx

* sudo apt-get install build-essential libtool zlib  (toolkit)
* sudo apt-get install nginx

## Nginx 相關目錄

/usr/sbin/nginx：主程序  
/etc/nginx：存放配置文件  
/usr/share/nginx：存放静态文件  
/var/log/nginx：存放日志

## MacOS install path

/usr/local/etc/nginx
